# ESP32

## Librarys
  * ArduinoHttpClient
  * ESP32_Arduino_Servo_Libary
  * HTTPClient
  * WiFi
  * WiFiClientSecure

All ESP32 libraries are under [```ESP32/lib```](ESP32/lib).

Make sure to put the libraries in the correct folder, they should be in:
```
C:\Users\"Your Name"\Documents\Arduino\Libraries
```
